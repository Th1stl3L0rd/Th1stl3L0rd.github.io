Flexbox Example

The following Flexbox layout contains various properties that alter the appearance of each column. In an xxl (1400+ px) screen each row will be contained on a single line. Depending on the screen size of the device used to view this layout, the elements may change their position and wrap either up or down a row. This style creates a staggered look that resembles a brick wall.
