Grid Example

The following Grid layout uses rows and columns to create a rigid structure. In an xxl (1400+ px) screen each row's columns will be contained on a single line. Depending on the screen size of the device used to view this layout, the columns may divide and move up or down a row to fit accordingly. By using the property col-sm-*, we can create specific widths for each column within their rows that will be responsive for mobile screens (567px or less).
